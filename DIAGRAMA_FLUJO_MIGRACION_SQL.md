## ?? Conceptos Clave Explicados

### **PRIMARY KEY**
- Identificador ˙nico para cada fila
- No puede haber duplicados
- Acelera b˙squedas

### **NVARCHAR vs VARCHAR**
- **NVARCHAR**: Unicode (soporta acentos, caracteres especiales)
- **VARCHAR**: ASCII (m·s r·pido pero sin acentos)
- Elegimos NVARCHAR para soportar nombres en espaÒol

### **IF NOT EXISTS**
- Evita errores si ya existe
- Es idempotente (se puede ejecutar m˙ltiples veces)

### **Par·metros SQL (@id, @Nombre, @Edad)**
- Previenen SQL Injection
- El motor de SQL los maneja de forma segura
